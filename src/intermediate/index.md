# gRPC Intermediate

[Hub](https://alibaihaqi.github.io/learning-docs/) › gRPC › Intermediate

## What you'll build

You extend the beginner Greeter with a **server-streaming** method that sends
several greetings, add proper **error handling** with gRPC status codes, and set
a **deadline** on the client call. Same `@grpc/grpc-js` stack, one new RPC.
