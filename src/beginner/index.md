# gRPC Beginner

[Hub](https://alibaihaqi.github.io/learning-docs/) › gRPC › Beginner

## What you'll build

A minimal gRPC service in Node.js: one method, `Greeter.SayHello`, defined in a
`.proto` file, implemented by a server, and called by a client. Start the
server, run the client, and see `Hello, Ada` printed. You'll use `@grpc/grpc-js`
and `@grpc/proto-loader` — no code generation step.
